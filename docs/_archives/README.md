# TSCP Framework

Transdisciplinary System Construction Principles.

## Vision
Un framework pour bâtir, analyser et faire germer des systèmes complexes par stratification ontologique et calcul tensoriel.

## Utilisation
- load <seed.jsonld> : Charge une graine séminale.
- grow --query <sparql> : Déploie l'isomorphe de connaissance à partir de la graine.
- analyse <id> : Calcule l'antifragilité et les métriques de phase.

## Modes
1. Sandbox : Offline, utilise des snapshots de triples stores.
2. Full Features : Hybride avec moteur d'inférence sémantique.
